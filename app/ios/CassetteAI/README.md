# CassetteAI — SwiftUI App Sources

App target for the CassetteAI companion app. Generated by AGENT-APP.

## Directory layout

```
CassetteAI/
  CassetteAIApp.swift      — @main entry point, AppSettings env inject
  RootView.swift           — TabView shell (Capture / Test Setup / Library / Settings)
  Audio/
    CaptureEngine.swift    — AVAudioEngine wrapper: 48 kHz float32 WAV, .measurement mode
  Views/
    DesignSystem.swift     — Colors, fonts, reusable gauge/badge components
    CaptureView.swift      — Hero capture screen: waterfall, level meter, record button
    DecodeView.swift       — Upload → poll → channel gauges → CRT boot reveal
    TestSetupView.swift    — Stage-A calibration flow: capture → POST /api/setup-test → tier badges
    LibraryView.swift      — Tape registry + QR deep-link explanation
    SettingsView.swift     — Backend URL, lab mode toggle, about
  Networking/
    BackendClient.swift    — async/await JSON client (decode, setup-test, manifest, calibration)
  Models/
    AppSettings.swift      — @AppStorage state (backendURL, labMode)
    TapeManifest.swift     — Codable models (TapeManifest, SetupTestResult, DecodeJob, DecodeResult)
```

## Run instructions

1. Run `xcodegen generate` from `app/ios/` (integrator step — do not run here).
2. Open the generated `CassetteAI.xcodeproj` in Xcode.
3. Build the CassetteDSP Swift Package first (it's a local dependency).
4. Select an iPhone 17.0+ simulator or device.
5. The backend defaults to `http://localhost:8765` — change in Settings tab.

## Key design decisions

- `AVAudioSession.Mode.measurement` — minimises OS input processing (AGC/AEC off).
- Voice processing explicitly NOT enabled (do not call `setVoiceProcessingEnabled(true)`).
- Float32 WAV written via `AVAudioFile` — never AAC.
- Real-time tap dispatches to a serial `DispatchQueue` — no allocation on audio thread.
- `CassetteDSP` package dependency wired for `LevelMeter`, `WaterfallProcessor`, `ChirpSync`,
  `SounderAnalyzer`, `SetupGrader` — `CaptureEngine` uses a fallback naive FFT until
  CassetteDSP is built; the integrator reconciles the API surface.
