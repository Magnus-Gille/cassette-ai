## Bruce Sterling — The Hacker Crackdown: License in Full

This is not a Creative Commons work. The license (from Project Gutenberg pg101.txt) states verbatim:

> "You can copy this electronic book. Copy the heck out of it… You can put the book on disks and give the disks away, as long as you don't take any money for it… But this book is not public domain. You can't copyright it in your own name. I own the copyright."

- No derivative terms. No version. No jurisdiction. Noncommercial redistribution is explicitly permitted, but this is a 1994 custom pre-CC license with no formal legal architecture behind it. Suitable for personal/gift distribution but not a formal CC grant.

## NC/ND/SA Quick Reference for Tape Distribution

| Clause | Meaning for cassette project |
|---|---|
| **NC (NonCommercial)** | Cannot sell the tape in a shop / for money. Gift distribution, personal use, educational use OK. |
| **ND (NoDerivatives)** | Must ship the exact unaltered text. An interactive reader that reflows, annotates, or restructures text MAY be considered a derivative — get legal advice or contact the author if uncertain. A straight verbatim text file is safe. |
| **SA (ShareAlike)** | Any derivative work (e.g. an annotated version) must use the same or a compatible CC license. The tape itself, if it incorporates SA material, should be distributed under the same license terms. |
| **No clause** | Code v2 (CC BY-SA): commercial distribution is legally permitted; SA applies to derivatives. |
